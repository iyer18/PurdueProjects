How To compile and Run!

1) Open .java file and change username and password
2) Enter the following commands:
	$ 	csh
	% 	source ~/.login
	% 	sqlplus
	SQL>	START drop.sql
	SQL>	START create.sql
	SQL>	START insert.sql
	SQL>	exit
	%	javac CourseManagementDB.java
	%	java CourseManagementDB
3) Now you may interact with the DataBase
